Derleme / Yükleme Uyarısı

Bu sayfayı doğrudan tarayıcıda (Live Server ile root klasöründen) açtınız veya derlenmemiş dosyaları yüklediniz. Tarayıcılar doğrudan .tsx dosyalarını okuyamazlar.

Çözüm Adımları:
  1. Hazır Tek Dosyayı Açın: Lütfen VS Code veya tarayıcınızda projenin içindeki dist/index.html dosyasını Live Server ile açın veya GitHub'a yükleyin. Bu dosya tamamen derlenmiş ve kullanıma hazırdır!
  2. Geliştirme Sunucusu: Projeyi yerelde çalıştırmak için terminalde sırasıyla npm install ve npm run dev komutlarını çalıştırın.
